Definisi Relasi

Relasi adalah hubungan antara elemen suatu himpunan dengan elemen himpunan lainnya.

domain himpunan asal (elemen dari X)
kodomain himpunan kawan (elemen dari Y)
range himpunan hasil dari relasi X ke Y
// notasi formal
R ⊆ A × B
(a, b) ∈ R ⟺ a R b
R⁻¹ = { (b,a) | (a,b) ∈ R }
S∘R = { (a,c) | ∃b : aRb ∧ bSc }
